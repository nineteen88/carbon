### Motion

When the `motion` prop is set to `false`, the motion will become static, and no animation will trigger.

<Canvas>
  <Story name="motion" story={stories.Motion}/>
</Canvas>

### Tracked

When the `tracked` prop is set to `true`, the spinner will become tracked, for specific use cases where wait/loading times
need to be predictable.

<Canvas>
  <Story name="tracked" story={stories.Tracked}/>
</Canvas>

### Animation time

The `animationTime` prop can also be set to any number, which will extend the animation time to that number in seconds.
